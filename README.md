# Dmitrii Petrukhin Academic Website

This repository contains a Hugo academic website for Dmitrii Petrukhin.

## Local Preview

Install Hugo Extended, then run:

```sh
hugo server --disableFastRender
```

The local preview is private to your machine. The site only goes public after changes are committed and pushed to GitHub.

## Deployment

GitHub Actions builds the Hugo site and deploys it to GitHub Pages on pushes to `main`.
